Select 选择器

基础组件-下拉选择框，模拟原生浏览器的下拉选择器

如何使用

  • 可以使用 v-model 双向绑定数据，value 只接受字符串和数字类型，组件会根据Option.value来选中数据。
  • 通过设置size属性为large和small将输入框设置为大和小尺寸，不设置为默认（中）尺寸。
  • 通过给Select设置属性disabled禁用整个选择器。
  • 通过给Option设置属性disabled可以禁用当前项。
<template>
    <div>
        <Select v-model="model1" style="width: 200px" size="large">
            <Option v-for="item in cityList" :value="item.value" :key="item">{{ item.label }}</Option>
        </Select>
    </div>
</template>
<script>
    export default {
        data () {
            return {
                cityList: [
                    {
                        value: 'beijing',
                        label: '北京市'
                    },
                    {
                        value: 'shanghai',
                        label: '上海市'
                    },
                    {
                        value: 'shenzhen',
                        label: '深圳市'
                    },
                    {
                        value: 'hangzhou',
                        label: '杭州市'
                    },
                    {
                        value: 'nanjing',
                        label: '南京市'
                    },
                    {
                        value: 'chongqing',
                        label: '重庆市'
                    }
                ],
                model1:''
            }
        }
    }
</script>

API

Select props

属性 说明 类型 默认值
value 指定选中项目的value值，可以使用 v-model 双向绑定数据 [String,Number] 空
size 选择框大小，可选值为large和small String -
disabled 是否禁用选择器 Boolean false
placeholder 选择框默认文字 String 请选择
